# cofounder.new — Website Rebuild (Var 2)

Static export of the cofounder.new marketing site built with React + Vite + Tailwind v4.

## Deployment

This folder is the static deployment artifact, auto-deployed to GitHub Pages via the GitHub Actions workflow in `.github/workflows/deploy.yml`.

Live URL: `https://tmq-gdteam.github.io/cofounder-website-recreate-var3-arvin/`

